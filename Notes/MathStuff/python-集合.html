Python 集合

好的，Python 的 set (集合) 提供了丰富的方法和一些相关的内置函数来操作集合中的元素。以下是一些最常用的：

1. 添加元素

  • add(element):

    • 向集合中添加单个元素。
    • 如果元素已经存在于集合中，则不执行任何操作 (因为集合元素是唯一的)。
    • 直接修改原始集合 (in-place)。
    my_set = {1, 2, 3}
print(f"添加前: {my_set}")

my_set.add(4)
print(f"添加 4 后: {my_set}") # 输出: {1, 2, 3, 4}

my_set.add(2) # 尝试添加已存在的元素 2
print(f"尝试添加 2 后: {my_set}") # 输出: {1, 2, 3, 4} (没有变化)

  • update(iterable) / |= 操作符:

    • 将一个可迭代对象 (如列表、元组、字符串或其他集合) 中的所有元素添加到当前集合中。
    • 相当于对每个元素执行 add()，同样会自动处理重复元素。
    • 直接修改原始集合 (in-place)。
    • |= 是 update 的运算符形式。
    set1 = {1, 2}
set2 = {3, 4}
list_to_add = [2, 5, 6]
string_to_add = "abc"

set1.update(set2) # 用 set2 更新 set1
print(f"更新 set2 后: {set1}") # 输出: {1, 2, 3, 4}

set1.update(list_to_add) # 用列表更新 set1
print(f"更新列表后: {set1}") # 输出: {1, 2, 3, 4, 5, 6} (2已存在，被忽略)

set1.update(string_to_add) # 用字符串更新 set1
print(f"更新字符串后: {set1}") # 输出: {1, 2, 3, 4, 5, 6, 'a', 'c', 'b'} (顺序不定)

# 使用 |= 操作符
set_a = {'x', 'y'}
set_b = {'y', 'z'}
set_a |= set_b
print(f"|= 操作后: {set_a}") # 输出: {'x', 'y', 'z'}

2. 移除元素

  • remove(element):

    • 从集合中移除指定的元素。
    • 如果元素不存在于集合中，会抛出 KeyError 异常。
    • 直接修改原始集合 (in-place)。
    my_set = {1, 2, 3, 4}
my_set.remove(3)
print(f"移除 3 后: {my_set}") # 输出: {1, 2, 4}

try:
    my_set.remove(5) # 尝试移除不存在的 5
except KeyError as e:
    print(f"移除 5 时出错: {e}") # 输出: 移除 5 时出错: 5

  • discard(element):

    • 从集合中移除指定的元素。
    • 如果元素不存在于集合中，它不会抛出错误，而是静默地不执行任何操作。
    • 直接修改原始集合 (in-place)。 这是 remove() 的“安全”版本。
    my_set = {1, 2, 3, 4}
my_set.discard(3)
print(f"Discard 3 后: {my_set}") # 输出: {1, 2, 4}

my_set.discard(5) # 尝试 discard 不存在的 5
print(f"Discard 5 后: {my_set}") # 输出: {1, 2, 4} (没有变化，也没有报错)

  • pop():

    • 从集合中移除并返回一个任意的元素。
    • 由于集合是无序的，你不能预测 pop() 会移除哪个元素。
    • 如果集合是空的，调用 pop() 会抛出 KeyError 异常。
    • 直接修改原始集合 (in-place)。
    my_set = {'a', 'b', 'c'}
removed_item = my_set.pop()
print(f"Pop 移除了: {removed_item}") # 输出可能是 'a', 'b' 或 'c' 中的任意一个
print(f"Pop 后的集合: {my_set}")    # 输出移除了那个元素后的集合

empty_set = set()
try:
    empty_set.pop()
except KeyError as e:
    print(f"对空集合 Pop 时出错: {e}") # 输出: 对空集合 Pop 时出错: 'pop from an empty set'

  • clear():

    • 移除集合中的所有元素，使其变为空集合。
    • 直接修改原始集合 (in-place)。
    my_set = {1, 2, 3}
my_set.clear()
print(f"Clear 后的集合: {my_set}") # 输出: set()

3. 集合运算 (通常返回新集合，也有对应的 in-place 更新方法/操作符)

这些操作用于比较和组合不同的集合。

  • union(other_set, ...) / | 操作符:

    • 返回一个包含两个 (或多个) 集合中所有唯一元素的新集合 (并集)。
    • | 是运算符形式。
    • 对应的 in-place 更新是 update() 或 |=。
    s1 = {1, 2, 3}
s2 = {3, 4, 5}
s3 = {5, 6}
union_set = s1.union(s2, s3) # 或 s1 | s2 | s3
print(f"并集: {union_set}") # 输出: {1, 2, 3, 4, 5, 6}
print(f"原始 s1: {s1}") # 输出: {1, 2, 3} (未改变)

  • intersection(other_set, ...) / & 操作符:

    • 返回一个只包含同时存在于所有指定集合中的元素的新集合 (交集)。
    • & 是运算符形式。
    • 对应的 in-place 更新是 intersection_update() 或 &=。
    s1 = {1, 2, 3, 4}
s2 = {3, 4, 5, 6}
s3 = {4, 5, 7}
intersect_set = s1.intersection(s2, s3) # 或 s1 & s2 & s3
print(f"交集: {intersect_set}") # 输出: {4}
print(f"原始 s1: {s1}") # 输出: {1, 2, 3, 4} (未改变)

# In-place update
s1.intersection_update(s2) # s1 &= s2
print(f"s1 交集更新后: {s1}") # 输出: {3, 4}

  • difference(other_set, ...) / - 操作符:

    • 返回一个包含存在于第一个集合但不存在于任何其他指定集合中的元素的新集合 (差集)。
    • - 是运算符形式。
    • 对应的 in-place 更新是 difference_update() 或 -=。
    s1 = {1, 2, 3, 4, 5}
s2 = {3, 4}
s3 = {5, 6}
diff_set = s1.difference(s2, s3) # 或 s1 - s2 - s3 (注意运算顺序)
print(f"差集 (s1 - s2 - s3): {diff_set}") # 输出: {1, 2}
print(f"原始 s1: {s1}") # 输出: {1, 2, 3, 4, 5} (未改变)

# In-place update
s1.difference_update(s2) # s1 -= s2
print(f"s1 差集更新后: {s1}") # 输出: {1, 2, 5}

  • symmetric_difference(other_set) / ^ 操作符:

    • 返回一个包含存在于两个集合中的一个，但不同时存在于两者中的元素的新集合 (对称差集，或异或)。
    • ^ 是运算符形式。
    • 对应的 in-place 更新是 symmetric_difference_update() 或 ^=。
    s1 = {1, 2, 3, 4}
s2 = {3, 4, 5, 6}
sym_diff_set = s1.symmetric_difference(s2) # 或 s1 ^ s2
print(f"对称差集: {sym_diff_set}") # 输出: {1, 2, 5, 6}
print(f"原始 s1: {s1}") # 输出: {1, 2, 3, 4} (未改变)

# In-place update
s1.symmetric_difference_update(s2) # s1 ^= s2
print(f"s1 对称差集更新后: {s1}") # 输出: {1, 2, 5, 6}

4. 比较和检查关系

  • issubset(other_set) / <= 操作符:

    • 检查当前集合是否是 other_set 的子集 (即当前集合的所有元素都在 other_set 中)。返回 True 或 False。
    • <= 是运算符形式。
    • < 运算符检查是否是真子集 (是子集，但两者不相等)。
    s1 = {1, 2}
s2 = {1, 2, 3}
s3 = {1, 2}
print(f"s1 是 s2 的子集? {s1.issubset(s2)}") # 输出: True
print(f"s1 <= s2? {s1 <= s2}")         # 输出: True
print(f"s1 是 s3 的子集? {s1.issubset(s3)}") # 输出: True
print(f"s1 <= s3? {s1 <= s3}")         # 输出: True
print(f"s1 是 s2 的真子集? {s1 < s2}")    # 输出: True
print(f"s1 是 s3 的真子集? {s1 < s3}")    # 输出: False (因为它们相等)

  • issuperset(other_set) / >= 操作符:

    • 检查当前集合是否是 other_set 的超集 (即 other_set 的所有元素都在当前集合中)。返回 True 或 False。
    • >= 是运算符形式。
    • > 运算符检查是否是真超集 (是超集，但两者不相等)。
    s1 = {1, 2, 3}
s2 = {1, 2}
s3 = {1, 2, 3}
print(f"s1 是 s2 的超集? {s1.issuperset(s2)}") # 输出: True
print(f"s1 >= s2? {s1 >= s2}")         # 输出: True
print(f"s1 是 s3 的超集? {s1.issuperset(s3)}") # 输出: True
print(f"s1 >= s3? {s1 >= s3}")         # 输出: True
print(f"s1 是 s2 的真超集? {s1 > s2}")    # 输出: True
print(f"s1 是 s3 的真超集? {s1 > s3}")    # 输出: False (因为它们相等)

  • isdisjoint(other_set):

    • 检查两个集合是否不相交 (即它们没有任何共同的元素)。如果没有任何共同元素，返回 True，否则返回 False。
    s1 = {1, 2}
s2 = {3, 4}
s3 = {2, 5}
print(f"s1 和 s2 不相交? {s1.isdisjoint(s2)}") # 输出: True
print(f"s1 和 s3 不相交? {s1.isdisjoint(s3)}") # 输出: False (因为它们有共同元素 2)

5. 其他常用操作

  • len(my_set): (内置函数)

    • 返回集合中元素的数量。
    my_set = {10, 20, 30}
print(f"集合大小: {len(my_set)}") # 输出: 3

  • element in my_set: (成员运算符)

    • 检查 element 是否存在于集合 my_set 中。返回 True 或 False。这是集合最常用的操作之一，效率非常高 (平均 O(1))。
    my_set = {'a', 'b', 'c'}
print(f"'b' in my_set? {'b' in my_set}") # 输出: True
print(f"'d' in my_set? {'d' in my_set}") # 输出: False

  • copy():

    • 返回集合的一个浅拷贝 (shallow copy)。修改新集合不会影响原始集合，反之亦然（但如果集合中的元素是可变对象，修改那个对象会同时影响两者）。
    original_set = {1, 2, [3, 4]} # 包含一个可变对象列表
copied_set = original_set.copy()

copied_set.add(5)
print(f"原始集合: {original_set}") # 输出: {1, 2, [3, 4]} (未变)
print(f"拷贝集合: {copied_set}")   # 输出: {1, 2, [3, 4], 5}

# 修改拷贝集合中的可变元素
copied_set.remove([3, 4])
copied_set.add([3, 4, 100]) # 添加一个修改过的列表
# 或者直接修改列表元素 (如果能拿到引用)
# for item in copied_set:
#     if isinstance(item, list):
#         item.append(100) # 这会同时影响 original_set

# print(f"再次原始集合: {original_set}") # 输出会受影响
# print(f"再次拷贝集合: {copied_set}")

  • 迭代: 可以使用 for 循环直接遍历集合中的元素。

    my_set = {'apple', 'banana', 'cherry'}
for fruit in my_set:
    print(fruit)
# 输出 (顺序不定):
# cherry
# banana
# apple

这些是 Python set 最核心和常用的方法与函数。掌握它们能让你高效地处理需要唯一元素或进行集合运算的场景。
