证明 (\sin x \cos x = \dfrac{1}{2} \sin 2x). 这里三角函数使用级数定义.

\end{ti}

\begin{s}

解:由定义, 令

[

\sin x = \displaystyle \sum_{n=0}^{\infty} (-1)^n \dfrac{x^{2n+1}}{(2n+1)!}

]

和

[

\cos x = \displaystyle \sum_{n=0}^{\infty} \dfrac{(-1)^n x^{2n}}{(2n)!}.

]

将原展开式写成形式

[

\sin x = \displaystyle \sum{n=0}^{\infty} \dfrac{\sin \dfrac{n\pi}{2}}{n!} x^n, \quad \cos x = \displaystyle \sum{n=0}^{\infty} \dfrac{\cos \dfrac{n\pi}{2}}{n!} x^n,

]

再利用柯西级数乘积法则, 得

[

\sin x \cos x = \displaystyle \sum{n=0}^{\infty} c_n x^n, \quad c_n = \displaystyle \sum{k=0}^{n} \dfrac{\sin \dfrac{k\pi}{2} \cos \dfrac{(n-k)\pi}{2}}{k!(n-k)!}.

]

因为

[

\sin \dfrac{k\pi}{2} \cos \dfrac{(n-k)\pi}{2} = \dfrac{1}{2} \sin \dfrac{n\pi}{2} + (-1)^{k+1} \dfrac{1}{2} \sin \dfrac{n\pi}{2},

]

并在初等公式

[

(x+y)^n = \displaystyle \sum_{k=0}^{n} \dfrac{n! x^{n-k} y^k}{k!(n-k)!}

中分别取 (x=y=1) 和 (x=-y=1) 导出

[

\dfrac{2^n}{n!} = \displaystyle \sum{k=0}^{n} \dfrac{1}{k!(n-k)!}, \quad \displaystyle \sum{k=0}^{n} \dfrac{(-1)^k}{k!(n-k)!} = 0,

[

cn = \displaystyle \sum{k=0}^{n} \dfrac{\sin \dfrac{k\pi}{2} \cos \dfrac{(n-k)\pi}{2}}{k!(n-k)!} = \dfrac{2^{n-1}}{n!} \sin \dfrac{n\pi}{2}.

\begin{p}[(菲赫金哥尔茨)数学分析原理, 第 2 卷, 条目447, 0.6, 证明题, 无穷乘积]

证明无穷乘积

[

F(x) = \prod_{m=1}^{\infty} (1 + q^m x) = (1 + q x)(1 + q^2 x)(1 + q^3 x) \cdots \quad (|q| < 1)

]

对于足够小的 ( x ) 可以展开成 ( x ) 的幂级数, 并确定这展开式的系数.

\end{p}

\begin{ti}

    无穷乘积可以取对数写成幂级数.

\end{ti}

\begin{s}

解:当 ( |x| < 1 ) 时乘积是收敛的并且有正的值;取对数我们得到

[

\ln F(x) = \displaystyle \sum{m=1}^{\infty} \ln(1 + q^m x) = \displaystyle \sum{m=1}^{\infty} \left( q^m x - \dfrac{1}{2} q^{2m} x^2 + \cdots \right).

]

特别当把 ( x ) 换成 ( |x| ) 与把 ( q ) 换成 ( |q| ) 时, 这级数收敛. 因此从 [445] 推出, (\ln F(x)) 在零的附近可以展开成 ( x ) 的幂级数, 而同样地[按照第 446 目的定理]也可以展开表达式

[

F(x) = e^{\ln F(x)}.

]

所以, 对于足够小的 ( x ), 我们有

[

F(x) = 1 + b_1 x + b_2 x^2 + \cdots + b_n x^n + \cdots

]

这里的系数 ( b_1, b_2, \cdots, b_n, \cdots ) 还需要决定. 如果从显然的等式

[

F(x) = (1 + q x) \cdot F(q x)

]

出发, 去实行这个计算比一切的方法都要简单, 利用展开式, 这显然可以写成

[

1 + b_1 x + b_2 x^2 + \cdots + b_n x^n + \cdots = (1 + q x)(1 + b_1 q x + b_2 q^2 x^2 + \cdots + b_n q^n x^n + \cdots)

]

的形式. 按照关于幂级数恒等的定理, 因此

[

b1 q + q = b_1, \quad b_2 q^2 + b_1 q^2 = b_2, \quad \cdots, \quad b_n q^n + b{n-1} q^n = b_n, \quad \cdots,

]

或者

[

b1 = \dfrac{q}{1 - q}, \quad b_2 = \dfrac{b_1 q^2}{1 - q^2}, \quad \cdots, \quad b_n = \dfrac{b{n-1} q^n}{1 - q^n}, \quad \cdots.

]

最后

[

b_1 = \dfrac{q}{1 - q}, \quad b_2 = \dfrac{q^3}{(1 - q)(1 - q^2)}, \quad \cdots,

]

[

b_n = \dfrac{q^{\dfrac{n(n-1)}{2}}}{(1 - q)(1 - q^2) \cdots (1 - q^n)}, \quad \cdots

]

\end{s}

\begin{p}[数学分析精选习题解析下, 4.48, 0.55, 证明题, 幂级数应用]

    设幂级数 (\displaystyle \sum_{n=1}^{\infty} a_n x^n) 的收敛半径大于零. 求证:

    \begin{enumerate}

        \item (\lim{x \to 0} \displaystyle \sum{n=1}^{\infty} a_n x^n = 0);

        \item 若 (a_1 \neq 0), 并且在原点的一个邻域内,

        [

        \left| \displaystyle \sum_{n=1}^{\infty} a_n x^n \right| \geq |a_1| |x| - 2x^2

        ]

        处处成立, 那么 (|a_2| \leq 2).

    \end{enumerate}

    \end{p}

    \begin{ti}

    \end{ti}

    \begin{s}

    证明:

    \begin{enumerate}

        \item 设 (\displaystyle \sum{n=1}^{\infty} a_n x^n) 的收敛半径为 (R > 0), 则级数 (\displaystyle \sum{n=1}^{\infty} a_n x^n) 在 ((-R, R)) 内的任一闭区间上一致收敛. 故

        [

        \lim{x \to 0} \displaystyle \sum{n=1}^{\infty} an x^n = \displaystyle \sum{n=1}^{\infty} \lim{x \to 0} a_n x^n = \displaystyle \sum{n=1}^{\infty} an \lim{x \to 0} x^n = 0.

        ]

        \item 用反证法. 若 (|a_2| > 2), 首先讨论 (a_2 > 2) 的情形.

        [

        \displaystyle \sum{n=1}^{\infty} a_n x^n = a_1 x + a_2 x^2 + x^2 \displaystyle \sum{n=3}^{\infty} a_n x^{n-2}.

        ]

        设 (a_2 = 2 + \epsilon), 其中 (\epsilon > 0). 应用第①小题的结果, 对上述的 (\epsilon > 0), 存在 (\delta > 0), 使得当 (0 < |x| < \delta) 时, 有

        [

        \left|\displaystyle \sum_{n=3}^{\infty} a_n x^{n-2}\right| < \epsilon.

        ]

        从而

        [

        \left|\displaystyle \sum_{n=1}^{\infty} a_n x^n\right| < \left|a_1 x + a_2 x^2\right| + \epsilon x^2.

        ]

        另一方面, 只要 (|a_2 x| < |a_1| \neq 0), 就成立

        [

        a_2 x^2 < |a_1||x|.

        ]

        这样, 当 (a_1 > 0) 时, 取 (x \in (-\delta, 0)), 当 (a_1 < 0) 时, 取 (x \in (0, \delta)), 从而

        [

        |a_1 x + a_2 x^2| = |a_1||x| - a_2 x^2 = |a_1||x| - (2 + \epsilon) x^2.

        ]

        故

        [

        \left|\displaystyle \sum_{n=1}^{\infty} a_n x^n\right| =     |a_1 x + a_2 x^2| + \epsilon x^2 = |a_1||x| - (2 + \epsilon) x^2 + \epsilon x^2= |a_1||x| - 2x^2.

        ]

        这就是说, 存在原点的右邻域 ((0, \delta)) 或原点的左邻域 ((-\delta, 0)), 使得

        [

        \left|\displaystyle \sum_{n=1}^{\infty} a_n x^n\right| < |a_1||x| - 2x^2,

        ]

        这显然与已知矛盾. 同理可证 (a_2 < -2) 的情形.

    \end{enumerate}

    \end{s}
