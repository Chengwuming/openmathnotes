核心：Python 的 dict 就是你要用的 Hash 表。

记住它的关键优势： 平均时间复杂度 O(1) 的 查找、插入、删除 操作。这是它解决很多问题的速度保证。

1. 必须掌握的 dict 核心操作 (语法层面):

  • 创建:

    my_dict = {}
# 或者带初始值
my_dict = {"a": 1, "b": 2}

  • 添加 / 更新: (如果 key 不存在则添加，存在则更新)

    my_dict['c'] = 3  # 添加新键值对
my_dict['a'] = 10 # 更新键 'a' 的值

  • 访问 (获取值):

    value = my_dict['a'] # 获取键 'a' 的值
# 注意：如果键 'a' 不存在，这行会抛出 KeyError 异常！

  • 安全访问 (避免 KeyError):

    # 方法一：使用 in 检查
if 'd' in my_dict:
    value = my_dict['d']
else:
    # 处理键不存在的情况
    value = None # 或者 0, 或者其他默认值

# 方法二：使用 .get() 方法 (更推荐！)
value = my_dict.get('d') # 如果 'd' 不存在，返回 None
value_with_default = my_dict.get('d', 0) # 如果 'd' 不存在，返回你指定的默认值 0

  • 删除:

    # 方法一：del 关键字
if 'c' in my_dict:
    del my_dict['c']

# 方法二：.pop() 方法 (返回被删除的值，键不存在会 KeyError)
if 'b' in my_dict:
    removed_value = my_dict.pop('b')

# 方法三：.pop() 带默认值 (键不存在时不报错，返回默认值)
removed_value = my_dict.pop('e', None) # 尝试删除 'e', 如果没有则返回 None

  • 检查 Key 是否存在: (非常常用！)

    if key_to_check in my_dict:
    print(f"{key_to_check} exists!")

  • 获取所有 Keys, Values, 或 Key-Value 对 (用于遍历):

    keys = my_dict.keys()     # 得到一个包含所有 key 的视图
values = my_dict.values() # 得到一个包含所有 value 的视图
items = my_dict.items()   # 得到一个包含所有 (key, value) 元组的视图

# 常用遍历方式
for key in my_dict: # 直接遍历 dict 默认是遍历 key
    print(key, my_dict[key])

for key, value in my_dict.items(): # 更清晰的方式
    print(key, value)

  • 获取大小 (元素数量):

    size = len(my_dict)

2. LeetCode 中 Hash 表 (Dict) 的核心应用模式 (实践层面):

以下是你在 LeetCode 中最常遇到的用 dict 解决问题的模式：

  • 模式一：计数器 (Frequency Counting)

    • 目标: 统计列表中每个元素出现的次数。
    • 标准 Python 实现: 
      counts = {}
my_list = [1, 2, 2, 3, 1, 1, 4]
for item in my_list:
    counts[item] = counts.get(item, 0) + 1
# counts -> {1: 3, 2: 2, 3: 1, 4: 1}
    • Pythonic 捷径: 使用 collections.Counter (本质上也是 hash map) 
      from collections import Counter
my_list = [1, 2, 2, 3, 1, 1, 4]
counts = Counter(my_list)
# counts -> Counter({1: 3, 2: 2, 3: 1, 4: 1})
# 可以像 dict 一样使用: counts[1] -> 3, counts[5] -> 0 (不存在的key返回0)
    • LeetCode 例子: "前 K 个高频元素", "字母异位词分组" (统计字符频率), "出现次数超过一半的元素"

  • 模式二：查找表 (Lookup Table / Seen Set)

    • 目标: 快速检查某个元素（或其“配对”元素）是否已经出现过。
    • 实现: 
      # 例子：两数之和 (LeetCode 1)
nums = [2, 7, 11, 15]
target = 9
seen = {} # 存 {数值: 索引}
for i, num in enumerate(nums):
    complement = target - num
    if complement in seen: # O(1) 查找！
        print(f"Found: index {seen[complement]} and index {i}")
        break
    seen[num] = i # 将当前数字和索引存入查找表
    • 关键: 把需要快速查找的信息（数值、索引、或其他属性）存入 dict，利用 O(1) 的 in 检查。
    • LeetCode 例子: "两数之和", "无重复字符的最长子串", "存在重复元素"

  • 模式三：分组 (Grouping)

    • 目标: 将元素按照某种属性（计算得出的 key）分到不同的组里。
    • 标准 Python 实现: 
      words = ["eat", "tea", "tan", "ate", "nat", "bat"]
anagram_groups = {}
for word in words:
    sorted_word = "".join(sorted(word)) # 计算分组的 key (排序后的字符串)
    if sorted_word not in anagram_groups:
        anagram_groups[sorted_word] = []
    anagram_groups[sorted_word].append(word)
# anagram_groups -> {'aet': ['eat', 'tea', 'ate'], 'ant': ['tan', 'nat'], 'abt': ['bat']}
result = list(anagram_groups.values())
    • Pythonic 捷径: 使用 collections.defaultdict(list) 
      from collections import defaultdict
words = ["eat", "tea", "tan", "ate", "nat", "bat"]
anagram_groups = defaultdict(list) # 指定默认值为 list
for word in words:
    sorted_word = "".join(sorted(word))
    anagram_groups[sorted_word].append(word) # 如果 key 不存在，自动创建空 list 并 append
result = list(anagram_groups.values())
    • LeetCode 例子: "字母异位词分组"

  • 模式四：缓存/记忆化 (Caching / Memoization)

    • 目标: 存储计算过的结果，避免重复计算（常用于递归或动态规划）。
    • 实现: 
      memo = {}
def fib(n):
    if n in memo: # 先查缓存
        return memo[n]
    if n <= 1:
        return n
    result = fib(n-1) + fib(n-2)
    memo[n] = result # 存入缓存
    return result
    • 关键: 用函数的参数（或其组合/哈希值）作为 dict 的 key，存储计算结果。
    • LeetCode 例子: 斐波那契数列，爬楼梯 (递归解法优化)，很多动态规划问题可以用字典做备忘录。

总结:

  • dict 是你的瑞士军刀，用于 O(1) 查找、计数、分组、缓存。
  • 熟练掌握 [] 访问/赋值、in 检查、.get() 安全访问、.items() 遍历。
